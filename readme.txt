Polaris Media Tools - A collection of visual effects and functions

== Description ==

Just a collection of some effects and functions:

* Panorama 360&deg; view shortcode
* Subpages shortcode for automated Subpages <ul></ul>
* Guestbook shortcode
* ID column in Posts/Pages Backendlist
* Function layout_content() for Contentecho in Design
* Polars Media AntiSpam API
*	Parentpage/Category Autolink Addon
*	jQuery Hover Submenu Addon
*	&amp;shy; Filter for title-Tag (Supports AiO-SEO Plugin)
*	Gallery CSS bugfix for different columns
* Hides Adminbar in Frontend for registered readers
* Added alternative Walker for wp_nav_menu (Lissaboneffekt)


== Installation ==

1. Upload the directories and files to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use Plugin via Admin-Menu PM Tools

== Frequently Asked Questions ==

= How to set the default settings =

Just send empty fields back in the formular and the default values will be set.
